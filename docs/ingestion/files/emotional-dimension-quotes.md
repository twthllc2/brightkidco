## KEY INSIGHTS

1. **Deadline Pressure** — Urgency is real and structural (preschool/PK enrollment). Every accident = threat to enrollment. Copy must match the panic frequency, not calm it prematurely.
2. **Social Pressure** — The main trigger is not the potty training itself but comparison to other kids. Core desire: "I don't want to be the only mom whose kid isn't potty trained." Mirror hooks perform best.
3. **Overwhelm** — These parents are depleted, not panicked. They need ease and relief, not more urgency info. The mess is the story — ruined furniture, endless laundry, physical exhaustion.
4. **Cross-dimension resonance** — Many quotes overlap dimensions (e.g., deadline panic also contains overwhelm). The primary dimension determines frequency selection; secondary dimensions inform tonal nuance.
