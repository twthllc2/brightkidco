## 1. Core Mechanism: Body-Signal Learning Layer™

**Proprietary system name:** 3-Layer Body-Signal Learning System™

**One-line statement:**
> Helps toddlers recognize when it's time to go by letting them feel a gentle wetness signal — the exact feedback pull-ups block.

**Expanded (ASD variant):**
> Supports body awareness without overwhelming sensory-sensitive children. The Body-Signal Learning Layer™ helps autistic kids connect body signals to potty timing through a gentle, predictable learning signal — not a harsh wet sensation.

### Core Science / Logic Chain

1. Child pees.
2. Child feels a gentle "uh-oh" sensation (not soaked, not uncomfortable).
3. Brain connects: "I peed" → "I feel it" → "I should use the potty."
4. Over time, awareness builds → child starts recognizing the signal BEFORE it happens.

### ASD-Specific Framing

For autistic kids, the core problem is **interoception** — the ability to notice internal body signals. Pull-ups block the one external feedback loop that helps the brain learn the connection. BrightKidCo restores that loop in a gentle, sensory-safe way.

---

## 2. The 3-Layer System (Detailed)

### Layer 1: Inner Body-Signal Layer
Soft cotton layer that allows the child to feel wetness immediately. Not soaked. Not uncomfortable. Just enough to trigger awareness. The exact opposite of moisture-wicking diapers. Preserves a gentle "uh-oh" sensation so the child can begin recognizing body signals.

### Layer 2: Smart Absorption Core
Absorbs up to **3x more than regular underwear** while preserving the learning signal. Protection without blocking awareness. Helps manage small accidents during the learning stage.

### Layer 3: Protective Outer Barrier
Prevents leaks onto clothes or furniture without killing the body-signal mechanism. **Leak-resistant, not leak-proof** — and that's intentional.

### 3-Way Positioning
- **Regular underwear:** Massive messes → parents give up / stress / meltdowns (ASD)
- **Pull-ups:** Zero sensation → kids don't learn
- **BrightKidCo™:** Controlled awareness → protection + learning together. "Not too dry. Not too messy. Just right for learning."
