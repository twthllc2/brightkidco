## 4. THE PRODUCT — BODY-SIGNAL LEARNING LAYER™

### 4.1 The 3-Layer System

```
LAYER 1: Inner Body-Signal Layer
  └─ Soft cotton, gentle "uh-oh" sensation (not soaked, not uncomfortable)
  └─ Opposite of moisture-wicking diapers

LAYER 2: Smart Absorption Core
  └─ Absorbs 3x more than regular underwear
  └─ Protection without blocking awareness

LAYER 3: Protective Outer Barrier
  └─ Leak-resistant (NOT leak-proof — intentional)
  └─ Prevents leaks onto clothes/furniture
```

**The Mechanism Logic Chain:**
1. Child pees → 2. Child feels gentle "uh-oh" sensation → 3. Brain connects: "I peed" → "I feel it" → "I should use the potty" → 4. Over time: awareness builds → child recognizes signal BEFORE it happens
