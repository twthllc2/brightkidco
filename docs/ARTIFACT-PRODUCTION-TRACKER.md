# Email Artifact Library — Production Tracker

> **Started: May 31, 2026**
> **Goal: 2,100 truly unique HTML artifacts for universal e-commerce email templates**
> **Status: Phase 1 — Building unique structure patterns**

---

## THE PROBLEM

Previous versions (v1-v5) used ~50 HTML structures with color/parameter variations.
User rejected this: "I want UNIQUE UNIQUE ideas, not color swaps."

Also: previous designs were "kawaii" themed (BrightKidCo specific).
Need: universal designs that work for ANY brand.

---

## THE PLAN

### Phase 1: 200 Unique HTML Structure Patterns
Each pattern = genuinely different HTML layout/creative concept.
NOT: same card with different border-radius.
YES: different way to arrange information visually.

### Phase 2: 10 Content Variations Per Pattern
Different headlines, quotes, stats, product names per instance.
200 patterns × 10 variations = 2,000 unique artifacts.

### Phase 3: Assembly
Combine into 300 templates × 7 artifacts = 2,100 total.
Each template = 7 unique artifacts (no repeats within template).

### Phase 4: Canvas + Polish
Figma-like canvas, tags, duplicate checker, theme system.

## DESIGN RULES (universal, not brand-specific)

### Structure rules:
- Each pattern = unique HTML structure (not CSS variation)
- Works with ANY of the 8 themes
- Mobile-first (single column, 420px max width)
- Email-client safe (tables for layout, inline styles)

### Visual rules:
- No brand-specific colors (use theme tokens)
- No brand-specific fonts (use system fonts + 1 display font)
- No brand-specific imagery (use placeholder descriptions)
- Works for: fashion, beauty, food, tech, kids, pets, home, fitness, etc.

### Quality rules:
- Each pattern must be visually distinct from all others
- No "same card with different border-radius" variations
- Different HTML structure = different creative concept
- Every pattern has a unique name and description
