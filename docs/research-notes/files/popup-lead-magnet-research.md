# Popup & Lead Magnet Synthesis — BKC Strategy Catalog

> Synthesized from RA5 R1-R4 research outputs. All recommendations are specific to BrightKidCo (Shopify + Klaviyo stack, autism/SPD parent audience).
> Compiled: May 2026

---

## 1. Popup Pattern Catalog

### Pattern 1: Multi-Step Quiz Popup (Highest Priority)
- **Type:** Multi-step interactive popup
- **Timing:** 10-second delay OR 30% scroll depth
- **Offer:** Personalized readiness/product score + free guide
- **Questions:** 5-7 questions (child age, sensory profile, biggest challenge)
- **Expected Conversion:** 8-12% (multi-step) to 30-50% (quiz completers)
- **Difficulty:** Medium — requires Octane AI or Prehook integration with Klaviyo
- **Why for BKC:** No competitor has a potty training readiness quiz. Parents actively search for readiness signs. Quiz captures high-intent leads while providing genuine value. Multi-step format converts 50-80% higher than single-step email gates.

### Pattern 2: Exit-Intent Lead Magnet Popup
- **Type:** Full-screen overlay triggered by cursor exit (desktop only)
- **Timing:** Fires when cursor moves toward browser close button
- **Offer:** Free downloadable guide (e.g., "The Parent's Guide to Interoception and Potty Training")
- **Questions:** Email only
- **Expected Conversion:** 3-7% of abandoning visitors
- **Difficulty:** Low — Klaviyo native support
- **Why for BKC:** No competitor uses exit-intent popups. Captures visitors who would otherwise be lost forever. Works on desktop; use time-delay (15-30s) fallback on mobile.

### Pattern 3: Age-Based Slide-In Popup
- **Type:** Slide-in from bottom-right corner
- **Timing:** 50% scroll depth on content pages
- **Offer:** Age-appropriate picks + 10% off first order
- **Questions:** "How old is your child?" (age bucket buttons) → email
- **Expected Conversion:** 12-18% (multi-step parent popup)
- **Difficulty:** Low — Klaviyo native multi-step forms
- **Why for BKC:** Age-based segmentation is the single highest-impact strategy for parent audiences. A parent of a 2-year-old and a 10-year-old have completely different needs. Asking about the child FIRST makes parents feel the brand cares.

### Pattern 4: Spin-to-Win Gamified Popup
- **Type:** Interactive wheel with prizes
- **Timing:** 8-10 second delay, triggered once per session
- **Offer:** Prizes: 10% off, free shipping, free gift with purchase
- **Questions:** Email required to spin; optional age question embedded
- **Expected Conversion:** 8.67% average (+132% vs. standard popups)
- **Difficulty:** Medium — requires third-party tool (Wheelio, OptinMonster) or custom build
- **Why for BKC:** Gamification works for parents when the game is quick (one spin) and prizes are genuinely useful. Embedding a segmentation question (child's age) inside the game captures data without feeling like a survey.

### Pattern 5: Welcome Discount Popup (Baseline)
- **Type:** Popup, 2-step (email first, SMS second)
- **Timing:** Exit intent (desktop), 15-second delay (mobile)
- **Offer:** 10% off first order with unique coupon code
- **Questions:** Step 1: email only. Step 2: SMS (optional). Step 3: coupon display.
- **Expected Conversion:** 3-7% of visitors
- **Difficulty:** Low — Klaviyo native, fully supported
- **Why for BKC:** Baseline email capture. Should run alongside (not instead of) quiz/lead magnet popups. Unique Shopify coupon codes prevent sharing.

### Pattern 6: Community-Framed Exit Popup
- **Type:** Exit-intent overlay
- **Timing:** Exit intent (desktop), scroll-depth fallback (mobile)
- **Offer:** "Join 10,000+ autism parents in our free community"
- **Questions:** Email for access
- **Expected Conversion:** 12% exit-intent conversion
- **Difficulty:** Low — Klaviyo native
- **Why for BKC:** Parents of children with autism are deeply community-oriented. Community-framed opt-ins outperform newsletter opt-ins by 20-30%. Community members had 4.2x higher LTV than non-members in case study data.

### Pattern 7: Embedded Footer Form (Persistent)
- **Type:** Embedded inline form in page footer
- **Timing:** Always visible (no trigger needed)
- **Offer:** "Stay in the loop — new products, parenting tips, exclusive offers"
- **Questions:** Email only
- **Expected Conversion:** 0.5-2% (lower but zero friction, always active)
- **Difficulty:** Very low — Klaviyo native embedded forms
- **Why for BKC:** Catches visitors who dismissed popups. Zero-friction persistent capture on every page. Works for blog readers and content consumers.

### Pattern 8: Post-Dismissal Teaser Tab
- **Type:** Small expandable tab at page edge
- **Timing:** Appears after popup dismissal
- **Offer:** "Still want 10% off? →" or "Take our quiz →"
- **Questions:** None (clicks expand into full popup)
- **Expected Conversion:** 1-3% additional capture from popup-dismissers
- **Difficulty:** Very low — Klaviyo native teaser feature
- **Why for BKC:** Second chance for visitors who dismissed the main popup. Non-intrusive, always visible, configurable position.

### Pattern 9: Cost Calculator Slide-In
- **Type:** Interactive widget or slide-in popup
- **Timing:** Product page engagement (image hover, add-to-cart)
- **Offer:** "See how much you'll save with BKC vs. Pull-Ups"
- **Questions:** Enter child's age → calculate monthly disposable cost → show savings
- **Expected Conversion:** 5-8% (interactive, high-value)
- **Difficulty:** Medium — requires custom calculator widget or third-party tool
- **Why for BKC:** bumGenius does a static version ($1,000+ savings). BKC can make it interactive and personalized. The savings math is compelling — $50-100/month on Pull-Ups vs. one-time BKC purchase.

### Pattern 10: Seasonal Flash Sale Full-Page
- **Type:** Full-page overlay with countdown timer
- **Timing:** Immediate or 5-second delay during sale periods
- **Offer:** Seasonal discount + countdown timer + email/SMS capture
- **Questions:** Email + optional SMS
- **Expected Conversion:** 5-10% (highest impact, highest friction)
- **Difficulty:** Low — Klaviyo native full-page forms + countdown timer
- **Why for BKC:** Countdown timers add +112.93% conversion lift (Sleeknote data). Use sparingly for BFCM, holiday sales, product launches.

2. **No competitor has a potty training readiness quiz** — single biggest missed opportunity

6. **Discount-based only** — no competitor offers educational value for email

## 3. Lead Magnet Ideas for BKC (Top 10)

### Tier 1: Highest Conversion Potential (30-50% opt-in)

**1. Sensory Product Finder Quiz**
- "What sensory tools does your child need?" — interactive quiz recommending products based on child's sensory profile
- Captures email after quiz completion, before full results
- Collects rich segmentation data (sensory type, age, challenges)
- Estimated conversion: 30-50% of quiz completers

**2. Potty Training Readiness Quiz**
- "Is your child ready for potty training?" — 5-7 questions about readiness signs
- Personalized readiness score + actionable next steps
- No competitor has this — parents actively search for readiness signs
- Estimated conversion: 30-45% of quiz completers

**3. Free Printable Visual Schedule Kit**
- Ready-to-use visual schedule cards for morning routines, school transitions, bedtime
- Autism parents universally need these — high perceived value
- Can be themed (back-to-school, summer, holidays) for seasonal refreshes
- Estimated conversion: 20-30%

### Tier 2: Strong Conversion Potential (15-30% opt-in)

**4. Interoception & Potty Training Guide**
- "The Parent's Guide to Interoception and Potty Training"
- Clinical authority positioning — no competitor owns this space
- Addresses the core challenge for autism parents in potty training
- Estimated conversion: 15-25%

**5. Weighted Blanket Sizing Calculator**
- Interactive tool recommending right weight/size based on child's age and weight
- Direct product tie-in — drives immediate purchase consideration
- Estimated conversion: 25-40%

**6. Sensory Diet Starter Guide PDF**
- 10-page guide with activities organized by sensory system (tactile, vestibular, proprioceptive)
- Written by or reviewed by an OT — therapist endorsement drives trust
- Estimated conversion: 15-25%

**7. IEP/504 Plan Checklist for Sensory Needs**
- Printable checklist parents can bring to IEP meetings
- Extremely practical — parents can use it immediately
- Estimated conversion: 15-25%

### Tier 3: Supporting Lead Magnets (10-20% opt-in)

**8. Meltdown vs. Tantrum: Parent's Decision Flowchart**
- Printable flowchart helping parents distinguish sensory meltdowns from behavioral tantrums
- Includes appropriate response strategies for each
- Estimated conversion: 10-20%

**9. Social Stories Template Pack**
- Editable social stories for common situations (dentist, new school, grocery store)
- ASD-specific, addresses daily challenges
- Estimated conversion: 10-20%

**10. Cost Savings Calculator: BKC vs. Disposables**
- Interactive calculator: enter child's age → see monthly Pull-Ups cost → see BKC savings
- bumGenius does static version ($1,000+ savings) — BKC can make it personalized
- Email gate: results delivered via email
- Estimated conversion: 10-20%

## 4. BKC Popup Recommendations — Specific & Actionable

### Immediate Deployment (Week 1)

**Popup A: Welcome Discount Popup**
- Type: Klaviyo native popup, 2-step
- Step 1: "Get 10% off your first order" + email field + CTA
- Step 2: "Want SMS updates too?" + phone field (optional) + "No thanks" link
- Success state: Unique Shopify coupon code displayed + "Check your email for your code"
- Trigger: Exit intent (desktop), 15-second delay (mobile)
- Targeting: New visitors only, all pages
- Frequency: Once per session, suppress for 30 days after dismissal

**Popup B: Teaser Tab (Post-Dismissal)**
- Type: Klaviyo native teaser
- Content: "Still want 10% off? →"
- Trigger: Appears after Popup A dismissal
- Click action: Expands into the welcome popup

**Popup C: Embedded Footer Form**
- Type: Klaviyo native embedded form
- Content: "Stay in the loop — new products, parenting tips, exclusive offers"
- Fields: Email only
- Placement: Footer of every page

### Phase 2 Deployment (Weeks 2-4)

**Popup D: Age-Based Multi-Step Popup**
- Type: Klaviyo native multi-step flyout
- Step 1: "How old is your child?" with age bucket buttons (0-12mo, 1-3, 3-5, 5-8, 8-12, 13+)
- Step 2: "Get age-appropriate picks + 10% off" + email field
- Trigger: 50% scroll depth on blog and collection pages
- Targeting: Visitors who dismissed or already saw the welcome popup
- Suppression: Existing subscribers

**Popup E: Exit-Intent Lead Magnet Popup**
- Type: Klaviyo native popup
- Offer: "Free Sensory Diet Starter Guide — 10 pages of OT-approved activities"
- Trigger: Exit intent (desktop), 30-second delay (mobile)
- Targeting: All visitors, sensory product pages prioritized
- Success state: "Check your email for your guide" + link to popular sensory products

### Phase 3 Deployment (Weeks 5-8)

**Popup F: Potty Training Readiness Quiz**
- Type: Octane AI or Prehook quiz integrated with Klaviyo
- Questions: 5-7 questions about child's age, behavior, readiness signs
- Flow: Quiz → personalized readiness score → email capture → Klaviyo profile properties synced
- Trigger: Embedded in navigation ("Take our quiz") + popup trigger on potty training pages
- Integration: Quiz answers sync to Klaviyo as custom properties for segmentation

**Popup G: Community-Framed Exit Popup**
- Type: Klaviyo native popup
- Offer: "Join 10,000+ autism parents in our free community"
- Trigger: Exit intent (desktop)
- Targeting: Visitors who viewed ASD/sensory content pages
- Success state: Welcome flow with community link + free resource

### Phase 4 Deployment (Weeks 9-12)

**Popup H: Cost Calculator Widget**
- Type: Custom widget or third-party tool (Wisepops/OptinMonster)
- Offer: "See how much you'll save with BKC vs. Pull-Ups"
- Mechanism: Enter child's age → monthly disposable cost → BKC savings
- Placement: Product pages, exit-intent on cart page
- Email gate: Results delivered via email

**Popup I: Spin-to-Win Gamified Popup**
- Type: Wheelio or OptinMonster spin-to-win
- Prizes: 10% off, free shipping, free gift with purchase
- Embedded question: "How old is your child?" (segmentation built into game)
- Trigger: 8-10 second delay
- Targeting: New visitors only

## 5. A/B Testing Roadmap for BKC

### Phase 1: Foundation Tests (Weeks 1-4)

| Priority | Test | Variable A | Variable B | Primary Metric | Run Time |
|----------|------|-----------|-----------|----------------|----------|
| 1 | Offer impact | 10% off coupon | No coupon (email only) | Signup rate | 2 weeks |
| 2 | Trigger timing | Exit intent | 10-second time delay | Signup rate | 2 weeks |
| 3 | Form type (mobile) | Popup | Flyout | Mobile signup rate | 2 weeks |

**Test 1 rationale:** Does the discount drive signups, or are parents signing up for value alone? If no-coupon performs within 20% of coupon, BKC saves margin.
**Test 2 rationale:** Exit intent captures abandoning visitors; time delay captures engaged visitors. Both matter — this reveals which converts better for BKC's traffic.
**Test 3 rationale:** Mobile is 60%+ of parenting traffic. Flyouts are less intrusive on mobile — does lower friction outweigh lower visibility?

### Phase 2: Optimization Tests (Weeks 5-8)

| Priority | Test | Variable A | Variable B | Primary Metric | Run Time |
|----------|------|-----------|-----------|----------------|----------|
| 4 | Discount framing | "10% off" | "Save $X on your first order" | Signup rate | 2 weeks |
| 5 | Social proof | "Join 10,000+ parents" | No social proof | Signup rate | 2 weeks |
| 6 | Image | BKC product photo | Lifestyle/child photo | Signup rate | 2 weeks |
| 7 | SMS step | Include SMS step 2 | Skip SMS, email only | Total subscribers | 3 weeks |

**Test 4 rationale:** Dollar amounts feel more concrete than percentages. Test which resonates with BKC's price point.
**Test 5 rationale:** Social proof is powerful for parents — "other parents trust us" validates the decision.
**Test 6 rationale:** Product photos show what they're buying; lifestyle photos show the outcome. Parents respond to outcomes.
**Test 7 rationale:** SMS capture adds friction but builds SMS list. Measure if the SMS step costs more emails than it gains SMS subscribers.

### Phase 3: Advanced Tests (Weeks 9-12)

| Priority | Test | Variable A | Variable B | Primary Metric | Run Time |
|----------|------|-----------|-----------|----------------|----------|
| 8 | CTA copy | "Get my discount" | "Join the family" | Signup rate | 2 weeks |
| 9 | Teaser | With teaser tab | Without teaser | Total signups | 3 weeks |
| 10 | Form style | Minimal/clean | Bold/colorful | Signup rate | 2 weeks |
| 11 | Mobile delay | 15 seconds | 30 seconds | Mobile signup rate | 2 weeks |
| 12 | Quiz vs. discount | Potty training quiz | 10% off welcome popup | Signup rate + downstream revenue | 4 weeks |

**Test 12 rationale:** The ultimate test — does educational value (quiz) outperform monetary value (discount) for BKC's audience? Measure not just signup rate but 30-day purchase conversion and average order value.

### Measurement Framework

| Metric | Target | Anti-Metric | Threshold |
|--------|--------|-------------|-----------|
| Form submission rate | >5% overall | Unsubscribe rate within 7 days | <3% unsub = quality signups |
| Email-only capture rate | >80% of submissions | — | — |
| SMS capture rate | >30% of email subscribers | — | — |
| Welcome flow open rate | >50% | — | — |
| First-purchase conversion | >15% of popup subscribers | — | — |
| Quiz completion rate | >60% of quiz starters | — | — |

**Run rules:**
- Minimum 1,000 impressions per variation for statistical significance
- Minimum 2 weeks per test (4 weeks for lower-traffic pages)
- Test one variable at a time
- Document every result for future reference

## 6. Klaviyo Implementation Notes

### Form Types to Use

| Form Type | Klaviyo Native? | BKC Use Case |
|-----------|----------------|--------------|
| Popup (Modal) | Yes | Welcome discount, exit-intent lead magnet, community popup |
| Flyout | Yes | Age-based slide-in, secondary capture for popup-dismissers |
| Embedded | Yes | Footer form on all pages, blog sidebar |
| Full-page | Yes | Seasonal sales, product launch waitlists |
| Teaser | Yes | Post-dismissal "still want 10% off?" tab |
| Quiz | No — need Octane AI or Prehook | Potty training readiness quiz, sensory product finder |

### Trigger Configuration

| Trigger | Desktop Setting | Mobile Setting |
|---------|----------------|----------------|
| Exit intent | Enable (highest converting) | NOT supported — use time delay fallback |
| Time delay | 8-15 seconds | 15-30 seconds (longer delay for mobile) |
| Scroll depth | 50% | 40% (parents scroll faster on mobile) |
| Page count | 2+ pages | 2+ pages |

### Targeting Rules to Enable

- **New vs. returning:** Show welcome popup to new visitors only
- **Already subscribed:** Auto-hide all forms for existing email/SMS subscribers
- **URL targeting:** Show sensory lead magnet on sensory pages, potty quiz on potty pages
- **Device type:** Desktop gets exit-intent; mobile gets time-delay or scroll-triggered
- **Traffic source:** Consider different offers for paid traffic (higher intent) vs. organic

### Frequency Capping

- Show each popup once per session maximum
- Suppress for 14-30 days after dismissal
- Never show popup to visitor who already converted
- Use Klaviyo's built-in suppression lists

### Multi-Step Form Best Practices

- Step 1: Email ONLY (highest conversion — asking for email + SMS simultaneously drops conversion 20-40%)
- Step 2: SMS (optional) with "No thanks" link
- Step 3: Success state with coupon code + "Check your email" message
- Each step should feel like progress, not an additional ask

### Coupon Integration

- Use unique single-use Shopify coupon codes (prevents sharing on coupon sites)
- Display code on-screen AND send in welcome email (dual delivery prevents loss)
- Set minimum spend coupling where appropriate ("$10 off $50+ orders")
- Track coupon redemption rate as popup success metric

### Flow Integration

Every popup must connect to a Klaviyo flow:
- **Welcome Flow** (5-email nurture sequence): triggered by popup submission
- **Abandoned Browse Flow**: triggered by product page views without purchase
- **Post-Purchase Flow**: triggered by first purchase — upsell, cross-sell, review request
- **Birthday/Age-Up Flow**: triggered by child's birthday — move to next age segment

### Klaviyo Limitations & Workarounds

| Limitation | Workaround |
|-----------|-----------|
| No mobile exit intent | Use time delay (15-30s) or scroll depth (40%) for mobile |
| No native quiz forms | Use Octane AI or Prehook → pass quiz answers as Klaviyo profile properties via API |
| No AI product recommendations in forms | Use static bestseller images; use Klaviyo's recommendation engine in post-signup emails |
| Limited conditional logic | Use multi-step forms with simple branching; use Klaviyo segments post-submission for personalization |
| No spin-to-win native | Use Wheelio or OptinMonster → integrate with Klaviyo via API |
| No revenue attribution on forms | Track manually via UTM parameters + Klaviyo revenue attribution |

### Third-Party Tool Integration Stack

| Tool | Purpose | Klaviyo Integration |
|------|---------|-------------------|
| **Octane AI** or **Prehook** | Quiz funnels (potty readiness, sensory product finder) | API sync — quiz answers become Klaviyo profile properties |
| **Wheelio** or **OptinMonster** | Spin-to-win gamified popups | API sync — email capture flows to Klaviyo lists |
| **Wisepops** | Advanced targeting (LTV, cart contents, AI recommendations) | API sync — standard Klaviyo integration |

### Segment-Based Popup Targeting

After collecting quiz/popup data, create Klaviyo segments for:
- **Child age segments:** 0-12mo, 1-3yr, 3-5yr, 5-8yr, 8-12yr, 13+
- **Diagnosis segments:** ASD confirmed, sensory processing (no diagnosis), ADHD, developmental delays, exploring
- **Challenge segments:** Finding sensory tools, managing meltdowns, sleep/routines, school/social
- **Profession segments:** Parent, therapist/OT, BCBA, teacher
- **Purchase intent:** Just browsing, looking for something specific, ready to buy

These segments drive personalized email flows, product recommendations, and content — far more valuable than the popup conversion rate alone.
