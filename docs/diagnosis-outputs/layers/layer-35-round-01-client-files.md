## 1. PHASE 1 vs PHASE 2+ BOUNDARIES

### What Phase 1 Covers (Autistic Parent Avatar ONLY)

Phase 1 focus is **exclusively** on the Autistic Parent Avatar with 4 sub-segments:

- **avatar_level1:** Verbal, 3-5 yrs, "almost trained" (Pre-K driven) — 70% hope tonality
- **avatar_level2:** Mixed verbal, 5-7 yrs, sensory + BCBA context (LARGEST segment) — 40% hope tonality
- **avatar_level3:** Non-verbal, 6-10+ yrs, dignity-first — 20% hope tonality
- **avatar_general:** Cross-level, includes diagnostic-pending, label-rejecters, sensory-different-non-ASD — 50/50

> - Deadline Parent Avatar

These avatars represent the three emotional dimensions that are explicitly OUT OF SCOPE for Phase 1:

## 4. GENERAL FALLBACK SCOPE RULES

### When to Use General Fallback (DOC 5)

- ~50% of real volume is General Fallback
- ALL Cart/Checkout/Browse mails are Cross-Level (DOC 5)
- ALL Post-Purchase Day 0-14 mails are Cross-Level (DOC 5)
- When level tag is unknown → Cross-Level (DOC 5)
- When unsure which level → default to General Fallback

### General Fallback Language Pattern

> "Whether...Or...If..." language pattern — addresses all levels simultaneously without committing to one.

### Migration Strategy

General Fallback audience transforms into Level segments through:
- Click/Reply/Purchase Hints
- Post-Purchase Level-Detection Mail (Day 14)
- Never guess — migrate through data

**Scope Rule:** Cross-Level mails (Cart, Checkout, Browse, Post-Purchase Day 0-14) ALWAYS use DOC 5 + DOC 1. Level-Specific mails use DOC 2/3/4 + DOC 1.

## 6. SEGMENTATION PHILOSOPHY — SCOPE PRINCIPLES

### Principle 1: Symptom-First, Never Level Labels

> **Verbatim from DOC6 (line 828-829):**
> "In mail copy NEVER say 'Level 2'. Instead describe symptoms. The tag is backend logic, not content."

### Principle 2: Tonality Calibration = Main Differentiator

Mails for Level 1 vs Level 3 don't differ in PRODUCT CONTENT, but in:
- Hope-Realism balance
- Forbidden Phrases (Level 3 stricter)
- Testimonial selection (age, complexity)
- Urgency level (Level 1 Pre-K OK, Level 3 NEVER urgency)

### Principle 3: Migration > Forced Segmentation

If unclear which level: use General Fallback, then migrate through Click/Reply/Purchase Hints. Never guess.

### Principle 4: DOC 1 Is Always Foundation

Regardless of avatar: DOC 1 (Master Foundation) is ALWAYS consulted. DOC 2-5 are sub-profiles, not substitutes.

### Principle 5: Content Avatar Definitions Are FIXED

Content avatar definitions are fixed (by Brand Strategist). Marketer decides which doc to consult based on avatar tag.

## 7. CROSS-LEVEL CALIBRATION RULES — SCOPE CONSTRAINTS

### Rule 1: Symptom Description Over Level Label

- NEVER: "If your child is Level 2 or Level 3..."
- YES: "If your child doesn't feel when they need to go..."

### Rule 2: 3-5 Recognition Anchors Per Mail

Every mail must have at least 3 specific situations spanning levels.

### Rule 3: Age Ranges Instead of Fixed Ages

- NEVER: "For 5-year-olds..."
- YES: "For kids 3 to 10+..."

### Rule 4: Verbal/Non-Verbal Inclusivity

Always include both verbal and non-verbal signal examples.

### Rule 5: Timeline Flexibility

- NEVER: "In 30 days your child will..."
- YES: "Some kids show first signals after 2 weeks. Others need 8 weeks. Both are normal."

### Rule 6: Success Metrics Low Bar

Define "success" as mini-wins, not "fully trained":
- "Look at the wet spot for the first time"
- "Pull at their pants in awareness"
- "Walk toward the bathroom (even after the accident)"
- "Stay dry 2 hours instead of 30 minutes"

## 8. FORBIDDEN SCOPE EXPANSIONS

### Avatar Mixing — ABSOLUTELY FORBIDDEN

- Deadline Parent Avatar content in Phase 1 Autism mails
- Social Pressure Mom Avatar content in Phase 1 Autism mails
- Overwhelmed at Home Mom Avatar content in Phase 1 Autism mails
- Any cross-pollination of avatar-specific RS files between Phase 1 and Phase 2+ avatars

### Voice Scope Boundaries

- Mary S. (Founder) in regular mails — ONLY in dedicated Founder Story mails (Phase 2+)
- Lena voice in Founder Story mails — Lena and Mary S. are separate identities
- Marketing voice in any customer-facing mail
- Corporate/professional tone

### Content Scope Boundaries

- Promising outcomes or timelines
- Using level labels in copy
- Comparing to neurotypical kids as standard
- Using "easy/simple/quick/guaranteed/miracle/cure/fix/solve"
- Using urgency manipulation ("only 24 hours!")
- Fabricating customer quotes
- Using Pre-K hooks for Level 2+3 audiences
- Using BCBA-aggressive language

### URL Scope Boundaries

- Using any URLs not specified in Brand Constants
- Using Facebook links (not used by brand)
- Using Trustpilot links (no page exists)
- Naming specific shipping carriers

## 9. RS-INTEGRATION SCOPE RULES

### What RS Files Are In Scope for Phase 1

- Autistic_Parent.pdf — Primary Autism Avatar research
- Autistic_Parent_G.pdf — Avatar G-file (deep psychological profile)
- Autistic_language_Bank.pdf — Verbatim quote collection
- Autistic_MIRO_m.pdf — Visual research insights (MIRO exports)

### What RS Files Are OUT of Scope for Phase 1

- All Deadline Parent RS files
- All Social Pressure Mom RS files
- All Overwhelmed at Home Mom RS files

### RS Usage Rules

- Mail text: 30-50% authentic verbatim language or tightly paraphrased
- Mail text: 30-50% Lena's voice (peer-to-peer, empathetic)
- Mail text: 10-20% mechanism/brand language (doc-based)
- Each mail must contain 2-4 authentic verbatim quotes from RS
- NEVER rewrite into "nicer English" — rough edges are trust anchor

### Anti-Fabrication Rule (NON-NEGOTIABLE)

> **Verbatim from Claude-Project-Instructions.txt (line 265-283):**
> "You may ONLY use verbatim quotes that exist in the RS PDFs provided in this project.
> You may NOT:
> - Invent customer quotes
> - Fabricate testimonials
> - Imagine 'what a mom might say'
> - Generate plausible-sounding but fake reviews
> - Adapt quotes so heavily they become fictional
> You MAY:
> - Lightly anonymize (change first names, change ages within range)
> - Combine themes from multiple real quotes into one representative statement (only if you note this internally)
> - Paraphrase for length, keeping core meaning and voice
> If you cannot find a real quote that matches what the mail needs, FLAG IT to Ayoub instead of fabricating."
