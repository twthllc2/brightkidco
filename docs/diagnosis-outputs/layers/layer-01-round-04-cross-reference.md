## 8. Gap Analysis — Features with No or Weak Cross-Reference

### CRITICAL GAPS (0 cross-references):

**F7: Easy Up & Down (Self-Use Support)**
- **Total references:** 0 across all 4 layers
- **Impact:** HIGH — This feature (child can pull underwear down/up independently) is a meaningful benefit for parents seeking independence, but it has NO cross-reference to any conversion lever, objection, failed method, or trust tier.
- **Why this matters:** The self-use benefit reduces supervision burden and builds child autonomy. In email flows, this could be a differentiator against pull-ups (which require parent handling). Currently invisible in the psychological architecture.
- **Recommendation:** Map F7 to:
  - L10-1 (Sensory Gap): "When the child can self-use, they build body awareness through ACTION, not observation"
  - L11-D2 (Maybe I'm the problem): "Your child CAN do this — it's built for their capability"
  - L13-T4 (Realistic Timelines): Independence builds gradually through daily practice

### SIGNIFICANT WEAKNESSES (1-2 references):

**F3: Protective Outer Barrier**
- Only maps to M7 (Naked Method). No lever, objection, or trust connection.
- **Recommendation:** Could map to L10-2 (Pull-ups worse: "protection without blocking signal"), B4 (GI issues: "leak protection during the learning process"), and T1 (mechanism specificity).

**F4: Leak-Resistant Design**
- Only maps to B4 and M7. Underutilized — this is the "less mess" promise that directly addresses Overwhelmed Mom.
- **Recommendation:** Map to L10-3 (reduces risk: "less mess during learning = less stress"), D3 (hope-risk: "you won't be cleaning disasters"), T4 (realistic: "accidents happen, we handle them").

**F8: Big Kid Underwear Feel**
- Only 3 weak references. The psychological "growing up" signal is undervalued.
- **Recommendation:** Map to D2 (child isn't a baby, reduces shame), T2 (peer voice: parents sharing "she finally WANTED to wear them").

**F9: Wash & Reuse**
- Only maps to C3 and M8. Cost savings is a secondary message but could be stronger in retention/upsell flows.
- **Recommendation:** Already reasonably placed — cost savings is a practical benefit, not a psychological lever. Current positioning (C3, M8) is appropriate.

**F11: Made in USA / Quality**
- Only maps to C1 and as peer trust support. Chemical safety is a hygiene factor, not a differentiator.
- **Recommendation:** Current coverage is appropriate for a trust signal that operates at the "denominator" level.

### STRONGEST FEATURES (10+ references):

**F12: ASD-Adapted Design** (20 references) — The highest cross-reference count. This is the most impactful product feature, touching all 4 layers heavily. It is the core differentiator for the autistic parent avatar.

**F1: BSL Layer** (17 references) — Second highest. The core mechanism drives mechanism specificity trust, counters the main objections, and solves the key failed methods.

**F6: Training-First Design** (13 references) — Third. The positioning of "learning tool not diaper" is deeply embedded in all layers.
