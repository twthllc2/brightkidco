## 2. CORE MECHANISM & TECHNOLOGY

### The 3-Layer Body-Signal Learning System™

| Layer | Name | Function | Source |
|---|---|---|---|
| **Layer 1** | Inner Body-Signal Layer | Soft cotton layer. Allows child to feel wetness immediately. Not soaked, not uncomfortable. Preserves gentle "uh-oh" sensation. Opposite of moisture-wicking diapers. | Produkt-Foundation, ProductData, BSL |
| **Layer 2** | Smart Absorption Core | Absorbs up to **3x more than regular underwear** while preserving learning signal. Protection without blocking awareness. Manages small accidents. | Produkt-Foundation, ProductData, BSL |
| **Layer 3** | Protective Outer Barrier | Prevents leaks onto clothes or furniture without killing the body-signal mechanism. **Leak-resistant, NOT leak-proof** — intentional. | Produkt-Foundation, ProductData, BSL |

### Mechanism Logic Chain

1. Child pees
2. Child feels gentle "uh-oh" sensation (not soaked, not uncomfortable)
3. Brain connects: "I peed" → "I feel it" → "I should use the potty"
4. Over time, awareness builds → child starts recognizing signal BEFORE it happens

**Source:** Unique-Mechanism-ProductData.txt, BSL.txt

### ASD-Specific Framing

> For autistic kids, the core problem is **interoception** — the ability to notice internal body signals. Pull-ups block the one external feedback loop that helps the brain learn the connection. BrightKidCo restores that loop in a gentle, sensory-safe way.

**Source:** Unique-Mechanism-ProductData.txt, BSL.txt
