## Skills Inventory

### 1. Brand Voice Definition (email-marketing/brand-voice-definition)

**Purpose:** Framework for defining, documenting, and deploying brand voice from raw source material.

**Key Techniques:**
- **Voice Dimension Scoring (1-5 scale):** Formal↔Casual, Serious↔Playful, Technical↔Simple, Authoritative↔Friendly, Salesy↔Educational, Short↔Long-form
- **Voice Pillars:** 3-5 pillars with do/don't examples per pillar
- **Tone Variation Table:** How voice adapts per channel (welcome, cart, post-purchase, SMS, service)
- **Keywords & Phrases:** Brand language (use/avoid) extracted from existing content
- **Raw Material Extraction:** Gather from website, social, existing emails, competitors

**Relevance to Kommentare Analysis:**
- HIGH. The voice dimension scoring framework can be applied directly to parent comments to calibrate "parent voice" — where parents fall on formal/casual, serious/playful, authoritative/friendly axes
- Voice pillars technique can be used to identify recurring themes/patterns in parent communication
- Keywords extraction applies directly to identifying parent vocabulary, recurring phrases, and taboo topics

## Application Plan for Layer 34

To analyze Kommentare and extract parent voice patterns:

1. **Classify** all comments using Reply Classification (Compliment/Question/Complaint/Suggestion/Spam)
2. **Score** each comment on the Voice Matrix 4 axes (Formal↔Casual, Serious↔Playful, Respectful↔Irreverent, Practical↔Aspirational)
3. **Extract** Core Vocabulary — most frequent words, phrases, emotional markers
4. **Map** comments to Journey Arc stages — where in the lifecycle is each parent?
5. **Apply** 5-Mode LLM Council to the full corpus — Contrarian (what's missing?), First Principles (what do parents actually need?), Expansionist (adjacent patterns?), Outsider (how would a non-parent read this?), Executor (what do we do with this?)
6. **Filter** using "So What?" test — which comments carry actionable signal vs. noise?
7. **Gap analysis** — what parent voice patterns exist that aren't covered by current brand voice?
