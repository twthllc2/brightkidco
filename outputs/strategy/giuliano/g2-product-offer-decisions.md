#### QW1 — Safety Certification (OEKO-TEX Standard 100)

| Detail | Value |
|--------|-------|
| **What** | Third-party safety certification for BKC's fabric & materials |
| **Why** | Peejamas already has Oeko-100 badge. Trust-sensitive ASD parents notice the gap. BKC claims PFAS/BPA/Chemical-Free but has no independent verification. |
| **Source** | Synthesis 17 §3.2, MASTER-RESEARCH-COMPENDIUM §2.5 |
| **Cost** | $1,000-5,000 (OEKO-TEX Standard 100 certification) |
| **Impact** | HIGH — trust signal that closes competitor gap |
| **✅ YES / ❌ NO** | **RECOMMEND: YES** |

#### QW2 — "Kids Helped" Social Proof Metric

| Detail | Value |
|--------|-------|
| **What** | Display "X children helped" counter (like Snugkins' "50,000+ Toddlers") |
| **Why** | Snugkins uses "50,000+ Toddlers" as primary trust metric. BKC has "100,000+ parents" but doesn't frame it as children helped. This is a conversion lever for skeptical L2 parents. |
| **Source** | Synthesis 17 §4.1 (Gap #3), MASTER-RESEARCH-COMPENDIUM §1.1 |
| **Effort** | Code change on site. Zero product development. |
| **✅ YES / ❌ NO** | **RECOMMEND: YES** |

#### QW3 — 2+2 Bundle Bridge at ~$54.99

| Detail | Value |
|--------|-------|
| **What** | Introduce a 2+2 bundle ($54.99 estimated) between 1 Pair ($34.99) and 3+3 ($79.99) |
| **Why** | The 2.3× price jump from $34.99→$79.99 is a psychological barrier. A bridge tier at ~$54.99 smooths to 1.6× → 1.5×. Also gives L3 a mid-path option. |
| **Source** | Synthesis 26 §4.2, Synthesis 06 §3.4 |
| **Risk** | May cannibalize some 3+3 sales. Requires new SKU + logistics. |
| **✅ YES / ❌ NO** | **RECOMMEND: EXPLORE (A/B test in Phase 2)** |

#### QW4 — "Graduation" Completion Underwear

| Detail | Value |
|--------|-------|
| **What** | A "Big Kid" cotton underwear pack sold as the graduation milestone for training completion |
| **Why** | Parents need an emotional closure point. No competitor offers this. Low effort (standard cotton underwear), high emotional impact for L1/L2 parents. |
| **Source** | Synthesis 17 §4.4 (White Space), MASTER-INGESTION-COMPENDIUM §1.1 |
| **✅ YES / ❌ NO** | **RECOMMEND: YES** |

### 3.3 Subscribe & Save Model Assessment

**FINDING: No viable subscribe-and-save model exists for BKC's core product.**

BrightKidCo sells reusable washable training underwear — a durable good with no consumable refill cycle. Unlike disposables (Pull-Ups at $50-100/month), BKC's product is a one-time purchase lasting months.

| Option | Viability | Detail |
|--------|:---------:|--------|
| **A: Replenishment Reminder** | ✅ RECOMMENDED | Email-based: "Your 3+3 arrived 75 days ago. Time to check fit?" Not a subscription. Captures repeat purchase without commitment. |
| **B: Accessory Subscription** | ❌ NICHE ONLY | Monthly delivery of Potty Targets + Charts requires new SKUs. Low viability. |
| **C: "Sizing Up" Pre-Pay** | ⚠️ FUTURE POTENTIAL | Child grows → needs S→M→L. Pre-paid program: $X/year for priority sizing. Requires sizing pipeline. |

**Recommended: Build Replenishment Reminder Flows (Option A). Do NOT build S&S for core product.**

*Sources: Synthesis 26 §5, Synthesis 32 §3*
