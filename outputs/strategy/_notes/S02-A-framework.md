## 1. FRAMEWORK SUMMARY

**The Core Insight (every email, every flow):**
> The parent does NOT have a potty training problem. They have a BODY SIGNAL TRANSLATION PROBLEM. [Source: MASTER-INGESTION-COMPENDIUM.md, Section 1, lines 99-101]

**The Universal Copy Formula (every email):**
1. Observation — Name what she's experiencing (specific scenario)
2. Validation — Absolve guilt ("It wasn't you")
3. Mechanism — Explain WHY neurologically (interoception gap)
4. Permission — Soft CTA / off-ramp ("Try it for 60 days. By your judgment.")
[Source: layer-09-voice-lena.md, Section 2]

**5 Conversion Levers (applied in non-negotiable order):**
- L1: Sensory Gap — "You haven't failed. You solved the wrong problem."
- L2: Pull-Ups Harm — "Pull-ups keep kids completely dry. That's the problem."
- L3: 60-Day Guarantee — "Try it for 60 days. By your judgment."
- L4: Lena Peer Trust — "Both my sons went through this. Here's what helped."
- L5: 49% SPARK Stat — "Almost 1 in 2 autistic kids this age aren't trained."
[Source: MASTER-DIAGNOSIS-COMPENDIUM.md, Section 4.1; layer-10-round-01-client-files.md]

**6 Cross-Level Calibration Rules (R1-R6) — non-negotiable for single-copy flows:**
- R1: Symptom over label — never name the level, name the experience
- R2: 3-5 recognition anchors per mail — so all levels see themselves
- R3: Age ranges over fixed ages — "kids 3 to 10+" not "5-year-olds"
- R4: Verbal/non-verbal inclusivity — include both modes in examples
- R5: Timeline flexibility — "some kids 2 weeks, others 8. Both normal."
- R6: Low-bar success metrics — mini-wins, not "fully trained"
[Source: GIULIANO-DEMANDS.md, Section 4]

**Trigger warning zones (absolute never):**
- "Easy", "Simple", "Quick", "Guaranteed", "Miracle", "Cure", "Fix", "Solve"
- Comparison to neurotypical kids as standard
- Urgency manipulation (countdown timers, fake scarcity)
- Level labels exposed in copy ("Level 1/2/3")
- Grief as sales tool (L3 — dignity violation)
[Source: MASTER-DIAGNOSIS-COMPENDIUM.md, Section 3.3; layer-09-voice-lena.md, Section 4-5; 08-email-copy-brief.md, Section 5]

## 2. THE 6-LAYER STRATEGY ARCHITECTURE

Every copy decision sits on 6 interrelated layers:

```
LAYER 1 — AVATAR LEVEL (Who is she?)
  ├─ L1 (Sarah, ~35%): Verbal 3-5, Pre-K driven, 70/30 hope, 2-7 day decision
  ├─ L2 (Lisa, ~40%): Mixed 5-7, BCBA context, 40/60 hope, 7-21 day decision
  ├─ L3 (Maria, ~25%): Non-verbal 6-10+, dignity-first, 20/80 hope, 14-30d
  └─ GF (~50% overlap): Unknown, variable, 50/50, symptom-language only
  [Source: 01-customer-avatar.md, Sections 3.2-3.5]

LAYER 2 — FLOW STAGE (Where is she in the journey?)
  Pre-Purchase → Conversion → Transactional → PP Early → PP Mid → Lifecycle
  [Source: 07-email-sequence-framework.md, Section 4.2]

LAYER 3 — EMOTIONAL PHASE (What's her energy?)
  Phase 1 (Slow Realization) → Phase 2 (First Attempt) → Phase 3 (Method Cycle) → Phase 4 (Resignation)
  [Source: GIULIANO-DEMANDS.md, Section 5]

LAYER 4 — CONVERSION LEVER (Which lever activates now?)
  L1 (Sensory Gap) → L2 (Pull-Ups Harm) → L3 (60-Day Guarantee) → L4 (Lena Peer) → L5 (49% Stat)
  [Source: 08-email-copy-brief.md, Section 3]

LAYER 5 — TRUST PYRAMID TIER (Which trust element?)
  Tier 1 (Mechanism) → Tier 2 (Peer Voice) → Tier 3 (Failure Ack) → Tier 4 (Timelines) → Tier 5 (Guarantee)
  [Source: layer-13-trust-pyramid.md, Section 1]

LAYER 6 — OBJECTION BEING ADDRESSED (What's her hidden question?)
  18 objections mapped to flow positions
  [Source: 08-email-copy-brief.md, Section 4; layer-11-objections.md]
```

**Copy Decision Flow:** For every email, define: Level → Stage → Phase → Primary Lever → Trust Tier → Primary Objection → Voice Calibration (Hope/Realism ratio, science density, CTA aggression) → Forbidden Words Check.

## 3. EMOTIONAL ARC PER FLOW

### Welcome Flow — 8-email emotional journey
```
E1: Curious-but-skeptical (code delivery, Lena intro)
    ↓
E2: "Holy shit, that explains it" (mechanism revelation — emotional tipping point)
    ↓
E3: Hopeful-but-cautious (60-day guarantee as hope shield)
    ↓
E4: Absolved (failure acknowledgment — "You didn't fail")
    ↓
E5: Informed (deep mechanism — Pre-K/science/dignity)
    ↓
E6: Connected (social proof — "I'm not alone")
    ↓
E7: Decisive (last call / permission not to buy)
    ↓
E8: Resolved (door stays open, farewell)
```
[Source: 07-email-sequence-framework.md, Section 6, Flow 01; GIULIANO-DEMANDS.md, Section 3]

### Recovery Flows (Cart/Checkout/Browse) — quick emotional arcs
```
Cart E1 (1h): Concern → Recognition (mechanism reminder)
Cart E2 (24h): Hesitation → Social proof (peer stories)
Cart E3 (72h): Fear → Release (guarantee + permission)
[Source: 07-email-sequence-framework.md, Section 6, Flow 02-04]
```

### Post-Purchase Education Arc (Days 0-21)
```
E1 (D0): Anticipation → Informed (first wear protocol)
E2 (D7): Uncertainty → Guided (what to look for)
E3 (D14): Discouragement → Normalized (regression is normal)
E4 (D21): Small hope → Validated (mini-wins celebrated)
[Source: 07-email-sequence-framework.md, Section 7, Flow 12]
```

### Winback Path A — 3-email re-engagement arc
```
E1: Acknowledged → Curious ("It's been a while. How are you?")
E2: Overwhelmed → Hopeful (new insight/mechanism reminder)
E3: Hesitant → Ready (guarantee + gentle CTA)
[Source: 07-email-sequence-framework.md, Section 7, Flow 21]
```

### FAQ/Objection Library — per-email emotional tension
```
Each email opens with the objection tension, then resolves:
E1: "Tried Everything" — Defeated → Absolved
E2: "Too Old/Severe" — Hopeless → Empowered (late-trainer stories)
E3: "GI/Withholding" — Scared → Supported
E4: "Cost" — Defensive → Valued
E5: "BCBA Compatibility" — Conflicted → Confident
E6: "Hope-Crash Fear" — Protected → Safe to try
[Source: 08-email-copy-brief.md, Section 4; layer-11-objections.md]
```
