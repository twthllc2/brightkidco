## Creative Strategy — Visual Direction

### The Big Idea

**"The signal that changes everything."**

Visual metaphor: A single gentle ripple spreading across water, representing the body signal "uh-oh" sensation radiating out from the Body-Signal Learning Layer. The ripple starts small (an accident, a moment of awareness) and expands outward (a pause, a look, a step toward the potty, a trained child). The visual communicates that change doesn't happen overnight, it starts with one signal, then another, building momentum.

The Big Idea reframes potty training from a behavioral checklist to a neurological awakening, the signal, not the method, is thero.

---

### Layout Architecture

**Hybrid Bento + Narrative Arc (adapted from email-cta-conversion-design §7.4, Pattern D: The Narrative Arc)**

A single-column mobile-first layout with visual breakpoints that create a guided reading journey. The layout uses generous negative space between sections to let each emotional beat land independently.

| Section | Visual Treatment | Layout Notes |
|---------|-----------------|-------------|
| Hero (Header + Subject area) | Brand wordmark centered, 48px padding top | Clean, no imagery — let the subject line and preview text breathe |
| Hook | Large pull-quote treatment — 24px italic serif, brand accent color left border | Pull-quote mimics the parent's voice, pulled out from body |
| Validation | Body text 17px, light gray background section divider | Subtle background tint signals a tonal shift |
| Mechanism | Bullet anatomy with subtle icon markers (checkmarks, not emoji) | 3-Layer visual mini-diagram using simple shape + text blocks |
| Social Proof | 3 testimonial cards, stacked vertically — quote in quotes, name/attribution below | Light card border, generous card padding (24px+) |
| Product | 3-column layer breakdown on desktop, stacked on mobile | Layer 1/2/3 with simple visual indicators |
| Guarantee | Centered block with subtle border treatment | "60 days. Your judgment." as a standalone moment |
| CTA | Full-width button on mobile, auto-width centered on desktop | Minimum 56px height, 18px bold text |

**Level 1 calibration:** Tighter spacing between sections than L2/L3 (24px vs 32-40px) to keep the email scannable for urgent Pre-K parents who are time-pressed. CTAs are larger (56px height vs 48px default). The Pre-K deadline is visually signaled without being explicit, a subtle timeline icon (calendar page, not clock) appears near the CTA only.

### Motion / Animation

**Client-constraint aware: designed for Klaviyo email (no JS, limited CSS support, GIF-safe)**

| Element | Motion | Technical Implementation | Notes |
|---------|--------|------------------------|--------|
| Signal Pulse Icon | 3 concentric arcs expanding outward — 2s cycle, ease-out | Animated GIF, 600×150px, transparent BG, 8 frames, <50KB | L1 cycle: 2s (faster than L2's 3s) |
| CTA Button Hover | (Desktop webmail only — Outlook fallback to static) | CSS :hover — background-color transitions from #2B6B76 to #D44A33 over 200ms | Only on email clients that support CSS hover |
| Testimonial Cards | Static — no animation | N/A | Motion on testimonials reduces perceived authenticity |
| Layer Illustration | Static GIF with subtle ambient pulse on Layer 1 inner ring | Animated GIF, 400×250px, 6 frames, <30KB | Pulse indicates "signal is active" |
| Section Dividers | Static only | 1px solid | No animation needed |

**Motion philosophy (per consolidated-design-skills-trimmed.md §Skill 3, impeccable Motion):**
"Don't animate CSS layout properties. Ease out with exponential curves. No bounce, no elastic."

All animations use `ease-out` timing. The signal pulse uses a slow exponential ease-out (not a staccato blink) to mimic the actual feeling of the "uh-oh" sensation, gradual awareness, not an alarm.

**Level 1 calibration:** The 2-second signal pulse cycle is the fastest in the system (L2: 3s, L3: 4s+). This the only place where animation cadence differs by level. The slightly faster pulse speaks to L1's Pre-K urgency without explicit urgency language.
