### DEEP MECHANISM

**Why the last 10% won't close with more practice or more prompts.**

The interoception signal for bladder fullness is subtle. For many autistic children, that signal is compressed — the brain registers it, just milliseconds too late. Timed methods and sticker charts hit a wall at that last 10% because they all assume the "I need to go" feeling arrives early enough to act on.

The **Body-Signal Learning Layer (BSL)** is designed for this specific problem. It provides gentle sensation that lasts — a 30-60 second "uh-oh" when an accident happens. Over time, the brain learns to associate the pre-event sensation with the post-event signal. The "almost trained" gap narrows.

## MOTION / ANIMATION

For animated HTML email (supported in Apple Mail, iOS Mail, and some Outlook app versions):

**1. Signal-pulse CTA glow**, The CTA button has a slow (3s duration, infinite loop) subtle pulse animation: the sage green background gently shifts +2% brightness and back. Creates micro-urgency without being aggressive. CSS: `@keyframes pulse { 0%,100% { filter: brightness(100%); } 50% { filter: brightness(102%); } }`

**2. Brain icon glow**, In the Signal Bridge illustration, the brain icon pulses a gentle soft white glow every 4 seconds. Visual metaphor: "the brain is waking up to the signal." Duration: 2s, ease-in-out.

**3. Quote fade-in**, The three testimonial quotes fade in sequentially (0s, +0.3s, +0.6s delays) when the email loads. Creates a "building evidence" feeling. Simple opacity + translateY(8px → 0).

**4. Signal arrow animation**, In the mechanism diagram, the dashed arrow in Panel 1 has a slow "energy travel" effect: a small dot moves along the arrow path from bladder to brain. Loop: 5s. Only in Panel 1 (the "normal signal" panel), Panel 2 is static (the "broken signal" panel). The contrast reinforces the message.

**5. Static fallback**, All animations gracefully degrade to static. The email is fully readable and conversion-optimized without any motion. Animation is enhancement, not requirement.

**Email client support for motion:**

- **Outlook desktop:** No animation support, static layers only ✓

**GIF fallback plan for critical motion assets:**
- The signal arrow travel effect is delivered as a 5s loop GIF (300px × 40px, <50KB optimized) for clients that strip CSS animation
