## SECTION 3: CREATIVE STRATEGY

### 3.1 Design Direction

| Element | Specification | Rationale |
|---------|-------------|-----------|
| **Color Palette** | Warm neutrals (#F5F0EB, #D4C5B2) + soft sage (#8FA88A) | No bold/aggressive colors. Calm, grounded, dignified. |
| **Typography** | Serif body (15-16px) for reading comfort. Sans-serif subject/preheader. | Serif slows reading pace — matches dignity-first cadence. |
| **Imagery** | No child faces. Abstract: soft wash of watercolor texture or blurred green landscape. | L3 parents are wary of stock photography of "happy trained children." Abstract = safe. |
| **Imagery Alternative** | Product flat lay — 1 Pair underwear on neutral surface, natural light. | Low-pressure product visual. No model. No "before/after." |
| **White Space** | Generous. 40-50px padding between sections. Single column. | Visual breathing room. Nothing crowded. Nothing urgent. |
| **Emphasis** | No bold + color combo on CTAs. Soft underline or muted button. | CTA should not scream. Soft presence aligns with permission-not-to-buy. |
| **Mobile** | Single column, 44px+ tap targets, body text 16px minimum. | >70% opens on mobile. Small text reads as "fine print" — violates trust. |

### 3.3 Design Rules (Non-Negotiable)

| Rule | Why |
|------|-----|
| No countdown timers or expiry badges | Any visual urgency triggers skepticism for L3 |
| No "🔥" or "⚡" emojis anywhere | Breaks the dignified tone immediately |
| No layered images or overlapping text | L3 parent reads slowly — clarity is respect |
| No sidebars or multi-column layouts | Creates visual "noise" — single column only |
| No bold/red CTA buttons | Colors associated with "sale" and "urgency" |
| No social proof counters in header | "16,511 reviews" feels like mass-market — not peer |
| No testimonials with full names/photos | Anonymity respects privacy of late-trainer families |
| Background color: very light warm (#F5F0EB) | Clinical white feels cold. Warmth = trust. |
