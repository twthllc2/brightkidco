## PART 2 — CREATIVE STRATEGY

### The Big Idea

**"The quiet path."**

This email is not a re-engagement push. It is a gentle arrival, returning to Maria's inbox not with urgency or new claims, but with the quietest form of communication: a shared truth the science has recently confirmed. The big idea is that progress for L3 children doesn't look like a breakthrough. It looks like a pause. A glance. A second of noticing something that was never noticed before. The email's visual and verbal architecture must communicate that this not about "training" in any conventional sense. It is about giving a nervous system information it has been missing, and letting biology do the rest.

The visual metaphor: a barely perceptible line that, on close inspection, carries the faintest ripple. Not a wave. Not a surge. A ripple. Because for Maria, a ripple is everything.

**Critical layout rule:** No element takes more than 40% of the vertical space. Content breathes. The layout should feel as if it is giving Maria room to decide, not filling every inch with persuasion.

**Mobile-first considerations (L3-specific):**

- The Ripple Line contraption scales to full container width minus 24px padding; minimum height of the SVG container remains 64px so the micro-undulation is still visible

### Custom Contraption

**Name:** The Ripple Line

**Concept:** A horizontal line-illustration spanning the full width (or centered in a 400px container) that tells the interoception story without words.

The line has two states, shown side by side:

```
 STATE A (left 50%) STATE B (right 50%)
 ────────────────────── ══╦══╤══╤══╤══╦══
 Flat, no perceptible Gentle, irregular
 variation. Representing undulation. Not a
 the slow to pick up signals smooth wave, more
 state: the signal is like a barely audible
 there but nothing signal that finally
 registers. registers. Representing
 a nervous system that
 has begun to receive
 information.
```

**Crucial design rules for this contraption:**
1. State B is NOT "full awareness." It is barely perceptible. The amplitude of the ripple is less than 8px. This crucial, for L3, success is not "trained." Success is "noticed something."
2. There is no arrow between State A and State B. No "before → after" framing. Just two possibilities shown as equally valid.
3. The line color is Brand-Green #039902 at 30% opacity on State A, moving to 60% opacity on State B, not full saturation. The change should be felt, not announced.
4. A subtle dot sits at the transition point, a visual anchor for "this where input began."

**Size:** Full width inside content area, max height 80px.
**Position:** Below section 3 (New Science) as a quiet visual rest between the mechanism explanation and the offer.

### Motion / Animation

**If rendered in AMP / interactive email (for subscribers who open on supported clients):**

The Ripple Line animates on a single slow loop:
- Duration: 6 seconds per loop
- State A (flat line) holds for 2 seconds
- Over the next 1.5 seconds, the line very subtly begins to show micro-variation, a tiny bump, a hesitation
- The ripple settles into the gentle undulation of State B over 1 second
- State B holds for 1.5 seconds
- Then the line slowly reverts to flat over 0.5 seconds
- Rest for 0.5 seconds, repeat

**Animation constraints:**
- No bounce easing. Use ease-in-out with a gentle curve.
- No scale transforms, the line does not grow or shrink
- No color flash, the opacity transition is the only color change
- No loop counter, it runs continuously but so subtly most subscribers won't consciously notice it

**Fallback for non-AMP clients (static):**
- Both states shown side-by-side as described in the Custom Contraption section
- The transition point dot serves as the static reminder that change is possible

**No animation applies to:**
- Text (no fade-in, no slide-up, text enters fully formed)
- CTA (no pulse, no glow, the CTA does not compete for attention)
- Product image (no zoom, no rotate)

**A/B Testing Considerations (Post-Warming):**
- A: No product image (Ripple Line only) vs B: Small product image
- A: "When you decide, it's here" vs B: "60 days, your judgment"
- Metric: click rate (not open rate, L3 opens may be high regardless)
- Minimum test duration: 14 days (L3 decision cycle is 14-30+ days)
- L3 sample size: given ~25% audience share and ~4-7% baseline conversion, a minimum of 5,000 L3-tagged subscribers per variant is recommended for statistical significance at 80% power (p<0.05)
- Guardrail metric: unsubscribe rate. If variant B (any change) increases L3 unsubscribe above the control, the test should be paused immediately, L3 parents unsubscribing is a signal of tone-deafness
- Secondary metric: reply rate (Lena reads all replies). A qualitative signal that matters more for L3 than click rate, replies from L3 parents indicate trust, not just action
- Do NOT test urgency elements (countdown, scarcity, "last chance"), these are forbidden for L3 under any conditions, even in a test cell
