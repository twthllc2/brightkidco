## 2. Strategy Box

**Big Idea:** The signal that changes everything — one gentle ripple spreading outward, proving that the body-signal learning mechanism works for L1 kids who are "almost there but not quite."
**Emotional Arc Position:** Cautious hope → Realistic acceptance → Proof through testimonials → Incremental hope. E4 is the social proof and realistic timelines email, showing L1 parents that inconsistent performance is the norm, not failure.
**Component Selection Rationale:** HERO-17 (Social Proof Hero) leads with a real parent quote to immediately anchor social proof. LETTER-01 (Paragraph Stack) provides clean body copy for the validation and mechanism sections. SP-01 (Yellow Quote Block) delivers the three L1-specific testimonials with visual warmth. EDU-9 (Tip Cards) presents the 3-layer system as actionable cards. CTA-5 (Product CTA) with Pre-K urgency framing matches L1's deadline-driven motivation.
**Differentiation Note:** L1 email uses higher chroma accents (coral #E87A5D), tighter spacing (24px section gaps), and a larger CTA (56px height) compared to GF's neutral warmth and L2/L3's calmer palettes. Calendar icon near CTA provides subtle Pre-K urgency without explicit deadline language.

## 8. Rendering Notes

- **Email client considerations:** Testimonial yellow quote blocks use table-based layout for Outlook compatibility. CTA button uses VML fallback for Outlook rendering. Signal pulse animation (GIF) falls back to static in Gmail/Outlook desktop. CSS hover on CTA only works in Apple Mail/iOS Mail.
- **Mobile responsive notes:** Single-column layout. CTA button goes full-width on screens < 600px. Hero image scales to 100% width. Section padding reduces to 16px on mobile. Tip cards stack vertically on mobile. Minimum 48px tap targets on all interactive elements.
- **Accessibility notes:** All images have descriptive alt text. Color contrast ratios: body text #4A3F3A on #FAF9F5 passes WCAG AA (~5.2:1). CTA white on teal passes WCAG AA (~7.8:1). Pull-quote coral on white passes WCAG AA (~4.8:1). Testimonial text on yellow badge passes WCAG AA.
- **Special instructions:** L1-specific: calendar icon (14×14px) + 3 countdown dots appear near CTA button as subtle Pre-K urgency easter egg. CTA button uses 56px height (vs 48px default) for L1 mobile urgency. Signal pulse animation cycles at 2-second rate (faster than L2's 3s). Lena's sign-off dash is slightly thinner for L1 continuity. Background is warmer (#FAF9F7) than GF's neutral (#FAF8F5). All urgency cues are implicit — no explicit "before Pre-K" deadlines.
