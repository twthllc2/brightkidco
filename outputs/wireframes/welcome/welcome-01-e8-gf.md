## 2. Strategy Box

**Big Idea:** The Open Field — a wide, calm landscape at dusk. Three weeks of information shared. Now the reader stands at a clearing. No fences, no arrows, no urgency. Just an open field with a gently lit path. The visual doesn't push, it invites.
**Emotional Arc Position:** End of 21-day arc. The reader has absorbed all information across three weeks. This is the moment of quiet decision — the only thing left is a choice, not a push.
**Component Selection Rationale:** HERO-3 (Permission Open) opens with space and warmth. LETTER-03 (Letter + Pull-Quote) delivers Lena's personal note with pull-quote emphasis. SP-01 (Yellow Quote Block) provides inclusive family quotes. EDU-14 (Quick Reference Card) recaps the four key learnings concisely. ProductShowcaseMedium shows product without overwhelming. CTA-4 (Gentle Invite) is a warm, permission-based CTA. TRUST-1 (60-Day Seal) removes risk. 20.3 (Gradient Band Variants) provides visual breathing room with warm dusk tones.
**Differentiation Note:** This is the only GF email with a migration anchor (quiz link). The constellation SVG contraption (7 interconnected dots) is unique to GF. Gold (#D4A76A) as primary CTA color — the only email in the entire welcome flow using gold. Slower animation pace than any other level (3-6s cycles). Asymmetric bento layout for recap section, not standard single-column.

## 8. Rendering Notes

### Email Client Considerations
- **Apple Mail / iOS Mail:** Full CSS animation support (constellation pulse, CTA glow, card fade-in, migration border glow). Georgia serif renders natively. SVG constellation fully supported.
- **Gmail (web + app):** CSS animations stripped. Static fallback for constellation (static SVG, 5 dots visible). CTA renders as solid gold button. Card fade-in replaced by static display.
- **Outlook (desktop):** VML background for recap cards (#F0EAE1). CTA rendered as VML button in gold. Constellation replaced by static image fallback. 2×2 grid collapses to single column.
- **Yahoo Mail:** CSS animations stripped. Layout degrades to single-column. All content readable. Constellation shows as static dots.

### Mobile Responsive Notes
- Single-column collapse at 480px breakpoint
- Signal dot constellation moves above hook headline
- Recap 2×2 grid stacks to single column with 12px gap
- CTA buttons become 100% width (minus 30px padding)
- Migration anchor card maintains full width
- H-padding reduces to 20px
- Font sizes: Hook 24px, Body 15px on mobile
- Price display reduces to 20px on mobile

### Accessibility Notes
- All text/background combinations pass WCAG AA minimum (4.5:1)
- CTA buttons pass AA — #2C2C2C on #D4A76A
- All images have descriptive alt text
- CTA buttons: minimum 44×44px tap target
- Semantic HTML: `<h1>` for hook, `<blockquote>` for quotes, `<ul>` for bullet points, `<article>` for recap cards
- Screen reader: "BrightKidCo, Welcome Email 8 of 8 — General"
- Dark mode: Background #1A1A1A, text #F5F5F5, gold CTA unchanged (sufficient contrast)
- All animations respect `prefers-reduced-motion`
- No element moves more than 8px total displacement

- This is email 8 of 8 in the welcome flow — the final GF email
