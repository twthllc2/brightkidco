## 2. Strategy Box

**Big Idea:** The Final Invitation — clear, warm, and unhurried. An open door with warm light spilling through, not a closing door, not a countdown. Completion, not expiration.
**Emotional Arc Position:** End of 21-day arc. The reader has received all information. This is the quiet resolution — the moment where the only thing left is a quiet choice.
**Component Selection Rationale:** HERO-3 (Permission Open) opens with warmth and space. LETTER-01 (Paragraph Stack) delivers Lena's final personal note with clean hierarchy. SP-01 (Yellow Quote Block) provides three L1 parent quotes for social proof. EDU-14 (Quick Reference Card) recaps the three-layer system concisely. ProductShowcaseFull gives full product visibility. CTA-5 (Product CTA) is a confident, single-action button. TRUST-1 (60-Day Seal) removes risk. UTIL-01 (Gradient Band) provides visual breathing room.
**Differentiation Note:** This is the only email in the welcome flow using a CTA Tunnel layout (Pattern A) — single focused CTA with minimal distraction. The entire layout funnels toward one decision. Warm but not soft color temperature. Georgia serif for letter voice. CTA button border pulse animation (heartbeat, not flash).

## 8. Rendering Notes

### Email Client Considerations
- **Apple Mail / iOS Mail:** Full CSS animation support (CTA pulse, layer hover). Georgia renders natively.
- **Gmail (web + app):** CSS animations stripped. Static fallback for CTA button (solid #2D5A3D). Georgia fallback to Times New Roman.
- **Outlook (desktop):** VML background for guarantee panel (#E8F0E8). CTA rendered as VML button. Georgia fallback to Times New Roman.
- **Yahoo Mail:** CSS animations stripped. Layout degrades to single-column. All content readable.

### Mobile Responsive Notes
- Single-column collapse at 480px breakpoint
- CTA button becomes 100% width (minus 30px padding)
- H-padding reduces to 20px
- Recap 3-column grid stacks to single column
- Social proof quotes maintain full width
- Product layer stack remains single column (already mobile-optimized)
- Font sizes: Hook 20px, Body 15px on mobile

### Accessibility Notes
- All text/background combinations pass WCAG AA minimum (4.5:1)
- CTA button passes AAA (7:1) — #FFFFFF on #2D5A3D
- All images have descriptive alt text
- CTA button: minimum 44×44px tap target
- Semantic HTML: `<h1>` for hook, `<blockquote>` for quotes, `<ul>` for bullet points
- Screen reader: "BrightKidCo, Welcome Email 8 of 8"
- Dark mode: Background #1A1A1A, text #F5F5F5, CTA green unchanged

### Special Instructions
- This is email 8 of 8 in the welcome flow — the final email
- L1 Sarah is deadline-driven (Pre-K): respect her time, be clear
- No countdown timers or urgency clocks in the design — urgency lives in copy only
- No product photography in the email itself — redirecting to site for imagery
- No gradient backgrounds — clean, resolved feeling
- The CTA button pulse is a quiet heartbeat, not a flashing signal
- Section divider dots (· · ·) between major sections create breathing room
- Lena's signature flourish is a subtle SVG stroke, not an actual signature image
