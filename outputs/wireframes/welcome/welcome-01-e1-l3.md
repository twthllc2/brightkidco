## 2. Strategy Box

**Big Idea:** A nervous system receiving a signal it never received before — depicted not as a breakthrough but as a quiet, almost imperceptible shift.
**Emotional Arc Position:** Mirror exhaustion → Validate dignity → Offer mechanism as neurological explanation → Quiet curiosity to continue. This is the first touchpoint for the most fatigued segment — Maria has been on this journey 6+ years and has zero trust left for marketing.
**Component Selection Rationale:** HERO-9 (Whisper Tone) opens as a whisper, not a headline — matches Maria's emotional register. LETTER-07 (Greeting Card) frames the text as a personal card, not a sales letter. TRUSTDEEP-15 (The Honest Negative) uses statistics as social proof — not testimonials (too salesy). EDU-1 (Definition Block) presents mechanism as calm fact. ProductShowcaseSoft gives the gentlest possible product mention. CTA-2 (Soft Link) is a text link, not a button. TRUST-4 (Dashed Policy Quote) frames guarantee as honest container.
**Differentiation Note:** L3 is the most visually restrained level. 60-70% negative space. No product photography. No illustrations of children. No emoji. No cards (too structural). The Signal Dot contraption is purely aesthetic — three quiet visual cues forming an invisible descending arc. The button is left-aligned, not centered, and uses Warm Charcoal instead of a bright color.

## 3. Creative Box

**Color Palette:**
- Primary: #C4A88B (Warm Sand) — Signal Dot accents, vertical line indicator (sparingly, 4-6 times total)
- Secondary: #A8B5A0 (Sage Moss) — Product layer lines, statistic separators
- Accent: #D4C5B2 (Pale Stone) — Watercolor circle, faded backgrounds
- Background: #FAF8F5 (Warm Ivory) — Full email body background
- Text: #3A3530 (Deep Clay) — Primary body copy (soft black, not #000)

**Typography:**
- Heading: Lausanne (or premium sans-serif), 24px (36px desktop), 300 (Light), line-height 1.3, letter-spacing 0.02em
- Body: Lausanne Text, 17px (20px desktop), 400 (Regular), line-height 1.6, letter-spacing 0.01em, color #3A3530
- Accent: Lausanne, 13px, 600 (Semibold), letter-spacing 0.08em, text-transform lowercase
- Statistics: Lausanne Display, 48px, 200 (ExtraLight), letter-spacing -0.02em

**Border Radius:** Cards: 12px, Buttons: 0px (no radius — flat), Images: 0px, Small: 4px
**Shadow:** Primary: none (L3 avoids structural shadows), Secondary: none
**Spacing:** Section gap: 48px minimum, H-padding: 24px, Card padding: 40px all sides, Element gap: 24px

## 8. Rendering Notes

- **Email client considerations:** All animations are CSS-only (Apple Mail, iOS Mail). Gmail and Outlook show static fallback. The Signal Dots are static circles — no animation required for them to communicate. The layout must work perfectly without any motion.
- **Mobile responsive:** All sections collapse to full-width with 20px side padding. The 50/50 hook split stacks vertically. Body text scales to 16px minimum (accessibility). All tracking preserved on mobile. CTA button goes full-width on mobile.
- **Accessibility:** Color contrast: #3A3530 on #FAF8F5 = 8.1:1 ratio (passes WCAG AAA). #4A4540 button with #FAF8F5 text = 6.8:1 (passes WCAG AAA). Statistics are text-based, not image-based. Alt text required for any images used.
- **Special instructions:** ZERO urgency in any element. No animated countdown timers. No pulsing buttons. No color-shifting backgrounds. No bouncing or elastic animations. All motion uses exponential easing (ease-out-quart via CSS cubic-bezier(0.25, 0.46, 0.45, 0.94)). The Signal Dot breathing animation (Dot #3) is 4s cycle, barely perceptible. CTA button is intentionally static — animation on CTA would signal urgency. The Easter Egg (signal-path vertical dotted line) is visible only on desktop (1200px+ viewport) and disappears on mobile.
