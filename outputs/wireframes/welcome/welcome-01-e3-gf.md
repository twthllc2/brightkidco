## 2. Strategy Box

**Big Idea:** "The Signal That Finally Arrives" — a delayed radio transmission struggling to connect, then reaching its destination, mirroring the 30-60 second body-signal feedback window.
**Emotional Arc Position:** Validation → Stat normalization → Mechanism clarity → Cautious hope (mid-flow, E3 of 8)
**Component Selection Rationale:** HERO-16 (Mechanism Tease) opens with the interoception mystery; LETTER-03 delivers the three-family hook with pull-quote emphasis; SP-03 (Stat Row) anchors the 49% figure; EDU-8 (Mechanism Diagram) explains interoception visually; ProductShowcaseFull reveals the product for the first time; CTA-5 (Product CTA) provides a clear next step; TRUST-1 (60-Day Seal) closes with safety; 20.3 (Gradient Band) creates visual transitions.
**Differentiation Note:** GF is the only email using three-family quote triptych. Unique Body-Signal Wave illustration. No level-specific language. Color warmth is neutral (not warmer like L1, not cooler like L3).
