Critical layout rule: No element takes more than 40% of the vertical space. Content breathes. The layout should feel as if it is giving Maria room to decide, not filling every inch with persuasion.

Mobile-first considerations (L3-specific):

- On mobile (320px-480px), cream background extends full width with 24px side padding

- All spacing between sections reduces by exactly 25% (48px → 36px, 40px → 30px, 32px → 24px), the reduced viewport naturally compresses breathing room

- The Ripple Line contraption scales to full container width minus 24px padding; minimum height of the SVG container remains 64px so the micro-undulation is still visible

- CTA text link remains 16px on mobile, never smaller, as shrinking the CTA reads as hiding it. 14px minimum across all breakpoints for body text.

- Lena sign-off and P.S. block should never split across a mobile screen boundary, use `white-space: nowrap` on key lines or sufficient padding to keep the signature visually intact

- On very small screens (under 360px), the left border on the Validation section reduces from 2px to 1px to preserve reading width

- The Hidden Easter Egg (footer line) must still render on mobile, it is not hidden by media queries. The 30% opacity is sufficient to make it unobtrusive, and mobile readers often scroll to the very bottom.

- Font size never increases on mobile (16-18px is already comfortable). There is no "zoom" or pinch-to-fill adjustment needed.

Name: The Ripple Line

Concept: A horizontal line-illustration spanning the full width (or centered in a 400px container) that tells the interoception story without words.

Crucial design rules for this contraption:

1. State B is NOT "full awareness." It is barely perceptible. The amplitude of the ripple is less than 8px. This crucial, for L3, success is not "trained." Success is "noticed something."

2. There is no arrow between State A and State B. No "before → after" framing. Just two possibilities shown as equally valid.

3. The line color is Brand-Green #039902 at 30% opacity on State A, moving to 60% opacity on State B, not full saturation. The change should be felt, not announced.

4. A subtle dot sits at the transition point, a visual anchor for "this where input began."

Size: Full width inside content area, max height 80px.

Palette:

Critical rules for L3:

- No bright yellow (#F5C84B). Yellow is high-energy and reads as "cheerful pressure" for Maria. It is forbidden in L3 email creatives.

- No high-contrast blocks. The difference between cream and white should be subtle enough that the eye rests, not jolts.

- CTA text is Dark Charcoal with Brand-Green underline. No button background. A filled button reads as "this the moment of decision", for Maria, the moment of decision must feel like her own, not compelled.

- The only place Brand-Green appears at full saturation is the "Lena" handwritten signature mark, a small visual trust anchor.

Primary Font: Questrial (weight 400 only, no bold variant used anywhere)

Fallback: system-ui, sans-serif

Size hierarchy:

Critical typography rules for L3:

- No bold weight anywhere. Bold is visual emphasis, it reads as insistence, which Maria's nervous system rejects. If emphasis needed, use a line break and white space instead.

- No italic. Italic reads as an aside or a whisper, for L3, the entire email is already at whisper volume.

- No ALL CAPS. All caps reads as urgency. Even "LENA" in the sign-off should be sentence-case: "Lena."

- Maximum body width: 520px in a 600px container. Lines should never exceed 70-75 characters to maintain readability for tired eyes reading on mobile.

- Letter-spacing increases slightly (0.02em) in the P.S. block to give it a quieter, more reflective feel.

If rendered in AMP / interactive email (for subscribers who open on supported clients):

The Ripple Line animates on a single slow loop:

- Duration: 6 seconds per loop

- State A (flat line) holds for 2 seconds

- Over the next 1.5 seconds, the line very subtly begins to show micro-variation, a tiny bump, a hesitation

- The ripple settles into the gentle undulation of State B over 1 second

- State B holds for 1.5 seconds

- Then the line slowly reverts to flat over 0.5 seconds

- Rest for 0.5 seconds, repeat

Animation constraints:

- No bounce easing. Use ease-in-out with a gentle curve.

- No scale transforms, the line does not grow or shrink

- No color flash, the opacity transition is the only color change

- No loop counter, it runs continuously but so subtly most subscribers won't consciously notice it

Fallback for non-AMP clients (static):

- Both states shown side-by-side as described in the Custom Contraption section

- The transition point dot serves as the static reminder that change is possible

No animation applies to:

- Text (no fade-in, no slide-up, text enters fully formed)

- CTA (no pulse, no glow, the CTA does not compete for attention)

- Product image (no zoom, no rotate)
